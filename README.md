A complete **Football League Management System** developed in **C++** as the final project for the **Computer Basics and Programming** course.

The application allows users to manage football teams, players, coaches, simulate matches, and generate league reports through a menu-driven console interface.

# 📖 Project Description

This project implements a complete football league management system using the C++ programming language.

The program enables users to:

- Create and manage football teams
- Add and manage players
- Assign coaches to teams
- Simulate football matches
- Generate league standings
- Display match history
- Manage league information through interactive menus

The project was implemented using only the concepts taught in the Computer Basics and Programming course.

# ⚽ Match Simulation Logic

The simulation works based on the following steps:

1. Select two teams.
2. Calculate the average strength of each team.
3. Apply a probability factor.
4. Determine the winner.
5. Save the match result.
6. Update league standings.
